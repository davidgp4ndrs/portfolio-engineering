# Engineering Portfolio | Embedded Systems & Software Development

A professional, high-performance portfolio built with a focus on technical clarity, modularity, and clean engineering principles. The UI features an "application window" aesthetic to reflect a multidisciplinary background in Electronics, Software, and AI.

## 🚀 Overview

This project serves as a digital CV and professional log. It is built using **Astro** for optimal performance (Static Site Generation) and **Bulma CSS** for a lightweight, flexbox-based responsive design.

## 🛠 Tech Stack

* **Framework:** [Astro 4.x](https://astro.build/)
* **Styling:** [Bulma CSS](https://bulma.io/) (No-JS CSS Framework)
* **Icons:** [FontAwesome 6](https://fontawesome.com/)
* **Language:** TypeScript
* **Environment:** Node.js
